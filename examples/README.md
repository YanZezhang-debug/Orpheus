# Orpheus 使用示例

本目录包含 Orpheus 的各种使用示例脚本。

## 线程参数测试

### Linux/macOS

```bash
cd examples
bash test_threads.sh
```

### Windows

```cmd
cd examples
test_threads.bat
```

### 功能说明

这些脚本演示了如何使用 `-t` 参数动态覆盖配置文件中的线程设置：

1. **测试 1**: 使用配置文件中的默认线程数
2. **测试 2**: 使用 `-t 16` 指定 16 个线程
3. **测试 3**: 使用 `-t 32` 指定 32 个线程

### 预期输出

运行脚本时，应该在日志中看到类似以下输出：

```
============================================================
Orpheus (奥菲斯) - 转录本可信性评估工具
版本: 0.2.0-dev (支持 TransDecoder)
============================================================
✓ 使用命令行指定的线程数: 32
  （将覆盖配置文件中的线程设置）
工作目录: /path/to/orpheus_output

步骤 1/1: TransDecoder ORF 预测
  同源搜索线程数: 32 (命令行覆盖)
```

### 前提条件

- 准备一个测试 FASTA 文件 `test_input.fasta`
- 确保已安装所需的外部工具（TransDecoder、Diamond 等）
- 如果启用了同源搜索，需要配置数据库路径
